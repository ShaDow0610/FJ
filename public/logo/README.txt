FD Logo Pack — Rouge / Noir / Or / Ivoire

Files:
- fd-logo-primary.svg   : Primary monogram (ivory on black, gold D stroke, red accent)
- fd-logo-inverse.svg   : Light version (black on ivory)
- fd-logo-outline.svg   : Transparent background (for overlays)
- fd-logo-emblem.svg    : Circular emblem (ring gold)
- fd-mark-compact.svg   : Compact mark for small sizes / favicon

Palette:
- Noir  : #0B0B0C
- Ivoire: #F6F5F1
- Or    : #C6A24A
- Rouge : #E01E37

Usage tips:
- Use the primary on dark sections; inverse on light sections.
- Outline works on any background; you can recolor fills/strokes in code.
- For PNG exports, open SVG in Figma/Illustrator or any converter and export at 512–1024 px.
